# MyAndroid

#### 介绍
Android开发组件和模块功能DEMO，干货满满。拿来即用

#### 软件架构
1.  app模块为DEMOs集
2.  每个XXDemo对应一个XX库模块



#### 安装教程

1.  git clone https://gitee.com/lc951/my-android.git


#### 使用说明

1.  IDE:Android studio 构建
2.  如果xxx库满足需要即可import module 选中目标xxx库目录即可导入到自己工程
3.  如有问题请提交wiki里
